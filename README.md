# reclaim-stats-exporter

Just a vibe-coded personal tool for `reclaim.ai` that converts calendar events to markdown.

## Usage
1. Open [app.reclaim.ai](https://app.reclaim.ai)
2. Click the extension icon
3. Hit **Extract Events**
4. Copy or download the generated markdown

## Privacy Policy

- **What information do you collect?**
  The extension reads calendar event titles and time ranges directly from the app.reclaim.ai DOM. It does not collect personal identifiers, server logs, or telemetry data.
- **How do you use the information?**
  Data is processed locally within your browser to calculate duration statistics and generate Markdown summaries. No data is stored or transmitted to external servers.
- **What information do you share?**
  No information is shared, sold, or transferred to third parties. All data remains strictly on your local device.
